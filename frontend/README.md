# ParkWave Frontend

React 18 + Vite frontend for the ParkWave parking management app.

## Setup

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
```

Requires the backend running on `http://localhost:5186` (Vite proxy handles CORS in dev).

See the [root README](../README.md) for full setup and API documentation.
